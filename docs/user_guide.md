# User Guide

## Purpose

This MVP Streamlit app estimates annual German income tax and annual net income using a simplified deterministic model.

## Inputs

- Annual gross salary
- Federal state
- Tax class
- Income year (2024-2026)
- Marital status
- Number of children

## Output

- Estimated annual income tax
- Estimated annual net income
- Short non-binding optimization hints

## Important note

The estimator is a simplified model for educational/demo purposes and does not replace official tax software or tax advice.
