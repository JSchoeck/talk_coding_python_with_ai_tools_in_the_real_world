# Coding Python with AI tools in the real world - an interactive live-coding session

This special event is based on cooperation between the [Python User Group Nürnberg](https://www.meetup.com/de-de/python-user-group-nuremberg/events/313635287/) and the [Nürnberg Data Science & Artificial Intelligence](https://www.meetup.com/de-de/nuernberg-data-science/events/313539876/) meetup groups.  
It will focus on AI code generation: We will highlight how to use AI to generate complete Python software projects.

A big thank you to codecentric AG for hosting this event with us! Many thanks for sponsoring the event, including the location and catering. 🙏  
You can expect exciting presentations and relaxed networking over snacks and cold drinks. 🍕🥤

## 🐍 Talk: Coding Python with AI Tools in the Real World

An interactive live-coding session — in a real-world setting, not a polished demo.  
We start from a concrete problem from the audience and solve it together using GitHub Copilot and modern LLMs. We prompt, iterate, debug, refactor, and occasionally get stuck — just like in everyday development.  
The AI will make mistakes, and we will fix them together. That's the point: learning how to work with these tools effectively, messiness included.

## 💡 What you'll learn

- **Getting started** — what you need to set up AI-assisted coding
- **Project structure** — how to structure a project for best AI results
- **Practical prompting** — prompting techniques for Python
- **Trust & verify** — when to trust AI and when to double-check
- **Real workflow** — how AI fits into everyday development

## 🛠️ Tools

- Python
- VS Code
- GitHub Copilot
- Coding-specialized LLMs: GPT Codex, Claude Opus, Gemini Pro
- Our creativity and problem-solving skills

## 👥 Target audience

All people interested in Python and/or coding with AI tools — regardless of experience level.  
Beginners and seasoned developers alike are welcome. Everything will be explained in an accessible and engaging way.
