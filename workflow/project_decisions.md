# Project decisions

## 2026-03-19

- Use a simplified deterministic tax model as an MVP placeholder and document assumptions clearly.
- MVP input fields include salary, federal state, tax class, income year, marital status, and number of children.
- MVP output is limited to estimated annual income tax and estimated annual net income.
- Include short optimization suggestions as text.
- Keep styling minimal with Streamlit defaults and subtle yellow/blue accents.
- Defer state-by-state comparison and detailed component breakdown to a later iteration.
- Update theme colors to a red-and-white palette for the app UI.
